# માર્ગદર્શિકાઓ

<!--
# Guides
-->

આ વિભાગમાં અમે માર્ગદર્શિકાઓનો સંગ્રહ રજૂ કરીએ છીએ જે તમને વર્ડપ્રેસ પ્લેગ્રાઉન્ડ સંબંધિત વિવિધ વિષયો સાથે કામ કરવામાં અને વધુ સારી રીતે સમજવામાં મદદ કરશે.

## [પ્લેગ્રાઉન્ડ દ્વારા મૂળ iOS એપ્લિકેશનમાં વાસ્તવિક વર્ડપ્રેસ સાઇટ કેવી રીતે મોકલવી?](/guides/wordpress-native-ios-app)

<!--
## [How to ship a real WordPress site in a native iOS app via Playground?](/guides/wordpress-native-ios-app)
-->

"બ્લોકનોટ્સ" તપાસો, જે વર્ડપ્રેસ પ્લેગ્રાઉન્ડ દ્વારા iOS પર નેટિવલી વર્ડપ્રેસ ચલાવનારી પહેલી એપ્લિકેશન છે. તે વેબએસેમ્બલી(WebAssembly) અને વર્ડપ્રેસ બ્લોક એડિટરનો ઉપયોગ કરીને સીમલેસ મોબાઇલ વેબ એકીકરણની સંભાવના દર્શાવે છે.

## [વર્ડપ્રેસ પ્લેગ્રાઉન્ડ સાથે તમારા ડેમો માટે સામગ્રી પૂરી પાડવી](/guides/providing-content-for-your-demo)

<!--

પ્લેગ્રાઉન્ડ દ્વારા તમારી થીમ અથવા પ્લગઇનનો સારો ડેમો પ્રદાન કરવા માટે, તમે તેને ડિફોલ્ટ સામગ્રી સાથે લોડ કરી શકો છો જે તમારા ઉત્પાદનની સુવિધાઓને હાઇલાઇટ કરે છે. તે કેવી રીતે કરવું તે જાણવા માટે આ માર્ગદર્શિકા તપાસો.
